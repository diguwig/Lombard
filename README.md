**📚 Документация приложения «Карточка клиента»**

1. Общее описание

Название проекта: Система управления клиентской базой (CRUD + Roles)

Версия: 1.0

Язык программирования: Python 3.x

GUI-библиотека: Tkinter

Автор: Иванов (ИСО-23, 1 подгруппа)

2. Назначение приложения

Приложение предназначено для ведения учёта клиентов с возможностью:

✅ Просмотра списка клиентов

✅ Добавления новых записей

✅ Редактирования существующих данных

✅ Удаления записей

✅ Разграничения прав доступа (Администратор / Пользователь)

3. Технические требования

Компонент: Требование

ОС: Windows 10/11, Linux, macOS

Python: Версия 3.8 и выше

Зависимости: tkinter (входит в стандартную библиотеку)

Разрешение экрана: Минимум 1600x800 пикселей

7. Функциональные возможности

7.1. CRUD-операции

Операция: Метод: Описание

Create: add_record(): Добавление нового клиента

Read: refresh_treeview(): Обновление таблицы

Update: update_record(): Редактирование записи

Delete: delete_record(): Удаление записи

7.2. Валидация данных

Поле: Правило валидации

Все текстовые поля: Не могут быть пустыми

Дата начала: Формат ДД.ММ.ГГГГ, год ≥ 2020

Уровень: Только из списка (Diamond/Platinum/Gold/Silver)

Email: Проверка на наличие @ (базовая)

7.3. Сообщения пользователю

Тип: Когда появляется

🟢 Успех: После успешного добавления/обновления/удаления

🟡 Предупреждение: При попытке редактирования без прав

🔴 Ошибка: При некорректном заполнении полей

🔵 Инфо: При смене роли

10. Обработка ошибок

Код ошибки: Ситуация: Решение

ValueError: Некорректный формат даты: Проверка формата ДД.ММ.ГГГГ

Empty Field: Пустое обязательное поле: Сообщение «Все поля должны быть заполнены!»

Access Denied: Нет прав на операцию: Сообщение «Доступ запрещен»

No Selection: Не выбрана запись: Сообщение «Выберите запись»

15. Примеры использования

15.1. Добавление клиента (Admin/User)

1. Заполнить все поля формы

2. Нажать кнопку «Добавить»

3. Получить подтверждение «Клиент добавлен!»

15.2. Редактирование клиента (Только Admin)

1. Выбрать клиента в таблице

2. Изменить данные в форме

3. Нажать кнопку «Сохранить изменения»

4. Получить подтверждение «Данные обновлены!»

15.3. Удаление клиента (Только Admin)

1. Выбрать клиента в таблице

2. Нажать кнопку «Удалить»

3. Подтвердить удаление в диалоге

4. Получить подтверждение «Запись удалена»

15.4. Смена роли

1. Выбрать роль в выпадающем списке (верхняя панель)

2. Получить уведомление о смене прав доступа

3. Кнопки редактирования/удаления активируются/деактивируются
